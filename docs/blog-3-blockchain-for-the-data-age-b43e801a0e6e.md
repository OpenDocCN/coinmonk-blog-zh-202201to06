# 区块链博客 03:区块链，面向数据时代

> 原文：<https://medium.com/coinmonks/blog-3-blockchain-for-the-data-age-b43e801a0e6e?source=collection_archive---------10----------------------->

![](img/90cf381f345381aa2386d3dc1e70a613.png)

自贸易开始以来，我们的经济一直在发展。我们从为生存而狩猎发展到今天的全球贸易。作为人类，我们总能找到降低彼此不确定性的方法，这样我们就能交换价值。如果我们回到狩猎采集经济时代，我们只是在村庄结构内进行贸易。我们有一些非正式的约束，但我们用暴力或社会反响来强制我们所有的交易。随着我们的社会变得越来越复杂，我们的贸易路线变得越来越远，我们建立了更多的正式机构，如货币银行、政府、公司。随着不确定性和复杂性的增加，这些机构帮助我们管理我们的交易，我们的控制力大大降低。最终，有了互联网，我们把这些相同的机构放到了网上。我们建立了像亚马逊、优步、Airbnb 这样的平台市场，只是更快的机构，作为中间人促进人类经济活动。这些机构是降低不确定性的工具，因此我们可以连接和交换社会中的各种价值。

![](img/9c340b9944cd2fc627d0c3f6cf58a0f6.png)

制度只是像宪法一样的正式规则，这些制度是让我们的经济车轮运转的润滑油。我们已经把我们的社会从农业和工业时代升级到信息时代(T2)，现在我们已经进入数据时代(T4)。在这个**数据时代**，机构利用他们收集的数据在经济结构中运作并提供更有用和更有效的服务。这就是为什么我们今天主要关心的是如何进一步降低我们在这些问题上的不确定性？在这个数据时代，我们还能相信中央机构吗？有没有更好的领养方式？如果我们有新的技术来进一步降低不确定性，不仅仅是政治和经济机构，像我们的银行，我们的公司，我们的政府，但是我们可以只用技术做到这一点。这就是区块链发挥作用的地方。因此，在了解区块链是什么之前，让我们先了解一下区块链是如何降低不确定性的，以及他们是如何承诺改造我们的经济体系的。

# 区块链是数据时代的解决方案？

在我们几乎所有的日常交易中，都面临着三种主要的不确定性，区块链可以在其中发挥作用。我们面临不确定性，比如不知道:

***1 -我们在和谁打交道？***

![](img/ea347ada700a20613daf1683e214c9b9.png)

比如我想在易贝上买一部二手智能手机。我要做的第一件事是查找我向谁购买。用户可信吗？他们有很好的评论和评级，还是根本没有个人资料？评论、评级、复选标记:这些是我们今天拼凑起来的关于我们身份的证明，用来降低我们在与谁打交道的不确定性。区块链允许我们创建一个开放全球平台，在这个平台上存储来自任何来源的关于任何个人的任何证明。这允许我们创建用户控制的便携身份。不仅仅是个人资料，这意味着你可以有选择地展示你的不同属性，以帮助促进交易或互动，例如，政府发给你的身份证。在现实世界和数字世界中拥有这种便携式身份意味着我们可以以一种全新的方式进行各种人类交易。

***2 对交易没有可见性。***

![](img/6479bbdfd6419a84f1a2824621cd1f40.png)

许多公司，尤其是那些生产智能手机等复杂产品的公司，面临的问题是他们要管理横向供应链上的所有不同供应商。所有参与制造产品的人都没有相同的数据库。他们不使用相同的基础设施，因此很难透明地看到一个产品随着时间的推移而发展。使用区块链，我们可以在不信任的实体之间创建一个共享的现实。我的意思是，网络中的所有这些节点不需要相互了解或相互信任，因为它们每个都有能力为自己监控和验证该链。所以我们可以用区块链来创造。我们可以创建一个分散的数据库，它具有与垄断相同的效率，而不需要实际创建中央权威。所以所有这些供应商，各种各样的公司，可以使用相同的数据库进行交互，而不需要相互信任。对消费者来说，这意味着我们可以有更多的透明度。就我们的可见性而言，这是一个全新的世界。

***3 出了事没有追索权。***

如果你不把智能手机发给我呢？我能拿回我的钱吗？区块链允许我们编写代码，绑定个人之间的合同，然后保证这些合同将在没有第三方实施者的情况下生效。例如，您正在为那部手机融资，但您不需要释放资金，直到您可以验证所有条件都已满足:您得到了那部手机。

我认为这是区块链降低我们不确定性的最令人兴奋的方法之一，因为这意味着在某种程度上我们可以瓦解机构和他们的执行。这意味着许多人类经济活动可以抵押化和自动化，并将许多人类干预推到边缘，即信息从现实世界转移到区块链的地方。因此，与其让我们所有的不确定性拖慢我们的脚步，并要求像银行、我们的政府、我们的企业这样的机构，我们实际上可以利用所有的集体不确定性，并利用它来进行更多、更快、更开放的合作和交流。

# 那么什么是区块链呢？

区块链只是一个在线分布式系统，可以存储信息，它是基于对所有人开放的互联区块。换句话说:它是一个可以存储信息的连接块链。

![](img/30bb8e060a80615969eabdf57d667efb.png)

区块链技术是一种去中心化的数据库，它通过点对点网络存储资产和交易的注册表。它基本上是谁拥有什么和谁交易什么的公共注册。

![](img/e5e5aef8781d441fabe465a2b20c7a19.png)

交易通过加密技术得到保护，随着时间的推移，交易历史被锁定在数据块中，然后这些数据块被加密链接在一起并得到保护。这就为网络上的所有交易创建了一个不可改变的、不可伪造的记录。该记录在使用网络的每台计算机上都被复制。

![](img/f29a91d1a20eb984f59d4873270b96d9.png)

在区块链，你可以把它想象成一个存储多种资产的开放基础设施。它存储了数字货币比特币等资产以及知识产权所有权等其他数字资产的监管、所有权和位置的历史。它可能是一份证书、一份合同、现实世界中的物品，甚至是个人身份信息。

![](img/96f52c43160d38f379390c722b448f13.png)

# 区块链技术还是婴儿吗？

嗯，我们需要在接下来的日子里做更多的研究。我不希望你有这样的印象，即区块链是一切的解决方案，事实是，这项技术还处于起步阶段，新的创业公司和现有的机构正在不断尝试。我们已经在各种金融和其他领域拥有了一些基于区块链的成熟技术。

从金融机构到科技公司、初创企业和大学，有成千上万的人在从事这项工作。其中一个原因是，这不仅仅是一场经济进化。这也是计算机科学的一项创新。区块链赋予我们技术能力，以一种全新的方式创造人类交流、货币交换、各种数字和实物资产，甚至我们自己的个人属性的记录。所以在某种程度上，他们变成了一个技术机构，拥有我们在社会中习惯使用的传统机构的许多好处，但它是以一种分散的方式做到这一点的。它通过将我们的许多不确定性转化为确定性来做到这一点。
