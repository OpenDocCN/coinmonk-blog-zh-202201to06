# 多边形(MATIC)

作为 Polygon Network 多链扩展解决方案的一部分，MATIC 充当以太坊令牌的燃料。Polygon 原名 Matic Network，旨在促进以太坊区块链与其他类似网络的互操作性。在过去的几年里，这个网站见证了巨大的流量增长。
