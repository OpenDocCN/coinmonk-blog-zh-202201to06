# 加密货币跳水——现在是购买的好时机吗？

> 原文：<https://medium.com/coinmonks/cryptocurrency-dipping-is-it-the-right-time-to-buy-523d23e8cb34?source=collection_archive---------39----------------------->

引起关注的新言论是迈克尔·塞勒的微观策略接近清算门槛，当这种情况发生时，BTC 几乎可以从几千美元缩减到几百美元。

听起来飘渺，如果你还没有买，你希望这个消息成真，然而，如果你属于其他群体，谁是 BTC 的支持者，并在多个低点买入，你真的想击败每个人谁希望 BTC 1 美元。但是现实地说，现在是购买 crypto 的时候吗？

# 市场条件和影响

[BTC](https://www.coindhan.com/trading/btcinr) 跌至过去两年未曾经历过的历史低点。甚至自 2021 年以来，BTC 一直徘徊在 3 万美元左右的水平，但最近暴跌至 2.2 万美元。在撰写本文时，ETH 还没有看到从 4300 美元的 ATH 跌至 1600 美元的不同命运。

所有这些发展都归因于宏观因素，如英俄战争、疫情刺激方案、通胀上升和美联储缩减购债规模。但是更大的画面显示，市场经历了过去几年未曾经历过的低迷。在上一个加密冬天，大部分资产几乎损失殆尽。比特币被注销了 80%，所以你可以简单地想象一下 [ALT 币](https://blog.coindhan.com/2022/04/11/how-to-pick-the-best-cryptocurrency-exchange-in-india/)会是什么情况。

# 那么，现在是买入的合适时机吗？

密码市场超越了逻辑推理和理性计算的领域。小市值可能意味着可能会出现前所未有的大起大落。我们已经在去年年底看到，市场确实跌至新低，并在 2021 年 11 月反弹至新的 ATH。然而，目前的情况有所不同，因为许多宏观因素影响了需求。最重要的是，Luna 和 Celcius 协议的惨败在一定程度上动摇了投资者的信任。

尽管事实如此，从比特币持有中明显可见的连锁活动表明，尽管下跌，积极情绪并没有恶化。BTC 经常在触底后反弹到新 ATH。所有这些都不可避免地会说，加密还没有走到尽头，协议和实现级别的发展表明，如果您希望在下一个减半到期时利用一些前所未有的收益，这可能是购买加密的最佳时机！
