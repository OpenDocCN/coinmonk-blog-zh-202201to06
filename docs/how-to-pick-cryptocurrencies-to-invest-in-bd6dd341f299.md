# 如何挑选加密货币进行投资？

你的选择越多，就越难做出好的选择。这是一句老话，它适用于加密选择，因为根据 [CoinMarketCap](https://coinmarketcap.com/) 的数据，526 家交易所有 19，626 种加密货币，总市值为 1.2T 美元。你现在可以看到，从我们拥有的数千种加密货币中挑选最好的非常困难。
