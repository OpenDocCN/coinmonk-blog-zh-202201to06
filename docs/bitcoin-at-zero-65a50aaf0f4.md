# 零比特币！

> 原文：<https://medium.com/coinmonks/bitcoin-at-zero-65a50aaf0f4?source=collection_archive---------18----------------------->

在我们意识到比特币真正的宇宙潜力之前，它的价格必须降到零。在比特币实现无价值的价格之前，它不会唤醒人类对其估值的应有意识。零是唯一能够表达价格和无价性同时性的先验价值——所有价值或意识从中散发的原始虚空。
