【2022年4月

随着春季的开始，四月的第一周通常会有大量的变化。然而，今年与以往不同，它的第一天可能会成为多个行业和3万多人最具变革性和洞察力的一周——至少对我来说是如此。本周以4月5日至6日举行的比特币解放大会开始，接下来将是在佛罗里达州迈阿密举行的比特币2022 大会。

《比特币解放》挤满了杰出的演讲者，以及以比特币为中心的节目，从主题演讲和小组讨论到现场演示和技术研讨会，涉及从分散金融(DeFi)到游戏(GameFi)、不可替代代币(NFT)和加密城市(CityCoins)、DAOs、基础设施改善、堆叠、风险投资(+ 比特币奥德赛)、&等等。

活动期间的对话分为四个核心主题:

  1. 在筹码上省钱和赚钱(STX)
  2. 投资&在堆栈上启动(STX)
  3. 在堆栈上花费和共享(STX)
  4. 交易&栈上融资(STX)

该活动有助于为任何人提供一个令人难以置信的协作环境，无论是行业、教育还是生态系统，都有能力走到一起，为比特币创新的未来奠定基础。仅仅过了几天，我还在回味那两天的兴奋和创新。幸运的是，我不仅有幸在活动期间发言(“a Odyssey Begins:Cash to Create on bit coin”)，还有幸主持了整整两天的活动——这两天我都很感激，并充当了灵感的迷你跳板。
