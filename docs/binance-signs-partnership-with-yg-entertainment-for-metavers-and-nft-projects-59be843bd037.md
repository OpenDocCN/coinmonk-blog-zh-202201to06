# 币安与 YG 娱乐公司签署合作协议，合作拍摄梅塔弗斯和 NFT 的电影

> 原文：<https://medium.com/coinmonks/binance-signs-partnership-with-yg-entertainment-for-metavers-and-nft-projects-59be843bd037?source=collection_archive---------19----------------------->
