# 辉瑞公司(PFE)

辉瑞公司(PFE 证券交易所代码:Pfizer)的股价一直在 50 美元关口徘徊。就在几周前，它还在 50 美元以下交易，现在略有回升。

抛开这一点，辉瑞应该被包括在内，因为它是疫苗的赢家。从这个角度来看，它今年的收入几乎肯定会超过 1000 亿美元。
