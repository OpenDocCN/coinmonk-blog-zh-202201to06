APY是通过使用投资额和投资保持活跃的时间段来计算的——以年度为基础。时间框架是一个基本要素，因为APY和简单回报之间的差异是复合的。下面是计算APY的公式:

APY= (1 + r / n)n — 1

这里， r 表示实际利率， n 表示评估复利的周期数。
