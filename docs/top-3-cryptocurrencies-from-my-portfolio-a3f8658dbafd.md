# 美元硬币(USDC)

美元(USD)与美国货币(USD Coin) 1:1 挂钩因此，流通中的每一张 1 美元钞票都由 1 美元储备(美国短期国债和现金的组合)支持。据该中心联合会称，USDC 是“由有执照的金融机构发行的”
