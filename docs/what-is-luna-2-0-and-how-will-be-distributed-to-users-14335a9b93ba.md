# 什么是 Luna 2.0

5 月 25 日，Terra Classic 用户批准了建立新的 Terra 链的提议。根据这个想法，Luna 的 genesis 发行版也将空投给用户。按照计划，Terra Classic chain 用户将获得基于 depeg 前后拍摄的快照的空投。
