# 卡尔达诺是什么？

你可能听说过第一代和第二代加密货币，比特币和以太坊。换句话说，Cardano 将自己描述为“第一个第三代加密货币”。像以太坊一样，它旨在提供一个平台，个人可以在这个平台上建立智能合约。
