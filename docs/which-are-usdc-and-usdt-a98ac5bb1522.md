# 什么是 USDC？

与美元挂钩的硬币被称为美元硬币(USDC)。被称为 USDC 的中心财团自 2018 年成立以来，一直是美元硬币的主要支持者。(由比特币基地和 Circle 领导)。该联盟声明的目标是生产一种稳定的数字货币，并开发一种环境，使金融合作伙伴、钱包、交易所和加密公司可以在当前的全球金融…
