# RSI =相对强度指数

作为一个动量指标，相对强弱指数(RSI)自带尺度。动量是价格变化的速度。一条从 0 到 100 的线表明市场处于超买和抛售状态。超买的市场价值超过 70，超卖的市场价值低于 30…
