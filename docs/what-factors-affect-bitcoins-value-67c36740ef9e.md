# 影响比特币价值的因素有哪些？

Source photo Unsplash.com

加密货币的价格波动有多种原因。需求和供应、投资者情绪、可用性、经济周期、采用率、功能、法律和新闻只是决定产品最终价格的诸多因素中的几个例子。与任何其他货币一样，比特币的价格也受到这些变量的影响，尽管它是一种众所周知的数字货币。
