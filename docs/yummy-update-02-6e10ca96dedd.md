煤渣的阶段是:

1)余烬
2)点火
3)火焰
4)地狱

余烬涉及其他项目 LP 的流失。那些项目不是死了就是被放弃了。另一个项目如何合并到 Cinder 中也有几个阶段，例如在 Cinder Token 下完全吸收或所谓的“软合并”，这意味着与 Cinder 建立某种形式的合作关系，但该项目仍然有自己的路线图和商业模式。

总体想法是大规模增加增长基金。
