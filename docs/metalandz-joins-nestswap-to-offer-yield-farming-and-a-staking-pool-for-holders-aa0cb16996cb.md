# Metalandz，这是怎么回事？

Metalandz 是一个基于 BNB 链的游戏平台。他们的愿景是提供一个深度沉浸式平台，玩家将在其中创建自己的不可替代令牌(NFT ),并通过玩 Poly Jungle 用我们的治理和效用令牌(METAZ)奖励他们的参与。
