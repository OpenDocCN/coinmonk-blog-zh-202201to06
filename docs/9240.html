什么是Vyper？

从官网上借鉴定义，

" Vyper是一种面向契约的pythonic编程语言，目标是以太坊虚拟机(EVM)"

这意味着vyper是智能契约编程语言领域中的python等价物。vyper是一种类似于python的简单易学的语言，所以让我们开始深入学习吧！

先决条件:

  • Python3

您可以通过执行以下命令轻松安装vyper:

python -m pip install vyper
