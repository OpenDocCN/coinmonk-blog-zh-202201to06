Notepad++中突出显示的实体语法

原文：https://medium.com/coinmonks/solidity-syntax-highlight-in-notepad-4419a7975634?source=collection_archive---------17-----------------------#2022-03-24

只是一个简短的帖子。我在寻找Notepad++的一些语法高亮插件，以便更好地处理Solidity代码，刚刚发现了这个关于这个的 StackExchange帖子。参考并感谢github用户 alin1popa 和 MegaManSE 提供了这样的模板。

所以，你要做的是:

  1. 下载。这些存储库中的一个xml文件。例如，Solidity.xml
  2. 将文件保存到% AppData % \ notepad++ \ userDefineLangs；
  3. 从记事本++中，转到语言->用户定义语言->定义您的语言>导入并指向文件；
  4. 关闭记事本++并打开。sol文件。
