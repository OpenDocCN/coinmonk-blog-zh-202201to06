## 符文令牌，非永久性损失保护，连续流动性池，等等！

![](img/4b67c702673fef37f75f8f31fe8a84d3.png)

THORChain 是一个建立在 Tendermint 和 Cosmos SDK 之上的流动性网络。使用 THORChain，用户可以在不同的区块链之间交换代币。符文令牌主要用作流动性池中的抵押品。
