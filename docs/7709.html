智能合同开发

让我们创建合同。sol:

// SPDX-License-Identifier: MIT

pragma solidity ^0.8.10;

contract MyContract {

string public myString = 'hello world';

}

我们来破解密码吧！

在第一行中，我们以机器可读的格式设置代码许可证

// SPDX-License-Identifier: MIT

在第二行中，我们设置了solidity编译器版本

pragma solidity ^0.8.10;

最后是主合同:

contract MyContract {

string public myString = 'hello world';

}

注意:可靠性代码必须在合同{}中

让我们编译代码

我们的代码编译成功，现在我们将部署到网络上

它会先询问是否允许访问我们的帐户来签署交易

当您接受它时，它将被启用部署

点击部署按钮，嘣！
