3 月 28 日，随着市场持续一周的暴跌继续，加密货币投资者情绪高涨，比特币(BTC)飙升至 48000 美元，以太交易超过 3400 美元。

根据 Cointelegraph Markets Pro 和 TradingView 的数据，在周一早盘围绕 47，000 美元的支撑短暂停顿后，下午的买盘飙升使 BTC 超过 48，000 美元，多头将 52，000 美元视为下一个目标。
