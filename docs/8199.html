什么是多边形？
Polygon是一个去中心化的以太坊扩展平台，使开发者能够以较低的交易费用构建可扩展的用户友好的dApps，而不会牺牲安全性。

为什么是多边形？
-大多数公司都在从 Instagram 、 Stripe 、 Addidas e.t.c
-更便宜的交易费
-全套以太坊-缩放解决方案
说到这里，让我们进入教程，我相信你已经安装了 MetaMask 作为浏览器扩展，并且你正在使用 hardhat 进行智能合约开发。

1。给元蒙版添加多边形访问https://mumbai.polygonscan.com/点击添加孟买网络按钮，弹出元蒙版，按照说明操作。

2。获取Matic令牌
Matic是为多边形网络供电的以太币令牌。
要获取Matic令牌，从元掩码中复制你的地址，粘贴到多边形龙头的钱包地址字段，点击发送给我MATIC 按钮。水龙头会送你5个测试自动令牌。

3。在炼金术上创建项目
炼金术是一个作为服务平台的节点；请使用https://www.alchemy.com/随意注册。
注册后，点击 +创建应用按钮。它弹出一个模态:
-填写你的DApp的名字
-填写关于你的dApp的小描述
-在链选择字段选择多边形。
-在网络中选择Polygon Mumbai，因为在本教程中，我们将部署到测试网络Polygon Mumbai。

4。复制项目HTTP关键字
创建项目后，您将重定向回仪表板，在那里您将看到新创建的项目，单击查看关键字按钮，并使用复制按钮复制 HTTP URL 。

5。更改hardhat.config中的Hardhat配置

我们最终到达那里，部署到多边形网络。在您的终端中运行下面的命令。

npx hardhat run scripts/deploy.js --network mumbai

恭喜你刚刚部署到多边形网络。👌
