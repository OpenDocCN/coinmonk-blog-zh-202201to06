Polygon 以前称为 MATIC network，是 2019 年开发的第二层扩展解决方案，旨在克服许多以太坊区块链限制，包括交易速度、吞吐量和天然气价格。

它是作为一个扩展解决方案创建的，但它很快成长为一个多用途的生态系统，引起了很多关注。它的原生硬币 MATIC 在 2019 年 IEO 繁荣时期出现在币安发射台。
