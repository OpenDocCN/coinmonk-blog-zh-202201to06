# 如何使用加密交易日志

如果你刚刚开始，你可以用纸笔或者 Excel 表格来记录你所有的交易。一种简单的方法是为您的:

1.  录入日期
2.  密码符号对
3.  入帐价格
4.  获利水平
5.  止损水平
6.  利润或损失
7.  利润/亏损百分比
8.  原因/注释

每次交易后，你应该把结果输入盈利或亏损部分，并写下你盈利或亏损的原因。改变每一行的颜色，以表明它是赢了(绿色)还是输了(红色)。
