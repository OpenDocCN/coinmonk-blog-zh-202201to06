## NFT 收藏品，比赛，奖品，等等！

![](img/cff54b66d74eaa3bdfa4989eacee1987.png)

Sorare 是一个总部设在 NFT 的玩家赚取游戏。玩家收集玩家的官方授权数字卡，并在比赛中使用这些卡来获得奖励。或者，他们可以交易这些卡来获利。
