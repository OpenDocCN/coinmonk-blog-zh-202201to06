# 多边形(MATIC)

以太坊多链可扩展性解决方案 Polygon Network 由以太坊令牌 MATIC 提供支持。为了使以太坊区块链和其他类似的网络更具互操作性，Polygon，最初被称为 Matic 网络，被创建。近年来，该网站的访问量有所增加。
