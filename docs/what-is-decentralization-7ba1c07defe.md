# 介绍

分散化是加密领域的常用术语。它是具有重要性质的基本哲学之一。对于那些刚接触加密的人来说，当人们抛出去中心化的财务、去中心化的应用等等时，可能会感到困惑。为了解释“去中心化”这个词的用法，我比较了集中和去中心化，解释了加密是如何使用这个术语的，并提供了加密去中心化类型的概述。读完这篇文章，你将成为一名分权专家！

# 集中式与分散式

为了解释去中心化，你需要知道中心化意味着什么，这样你才能知道去中心化意味着什么。集中化是指有一个你信任的中央权威来跟踪某事或为你做决定。集权意味着只有一个拥有权力的权威。在政治上，集权的最终结果是独裁。政治权力集中在一个人手中。对于金融来说，这意味着有一个中央机构来跟踪你委托的资金，比如中央银行。美联储银行是美国银行系统集中的中央银行。

既然你知道了集权是什么意思，现在我们可以谈谈分权了。对于上面的政治权力例子，这意味着一个国家正在从中央集权制(独裁)走向分权制(民主)。政治权力现在被分配给多个当局，而不是一个。对于银行业来说，你的钱不是委托给一个机构，而是分配给多个机构。想象一下，如果你的账户余额被银行的其他用户，即多个权威机构验证。这将是一家分权银行。注意，去中心化的关键是权力是分散的。例如，集中式银行是分布式的，即它们在世界各地都有分行和子公司，但它们在权限上仍然是集中的。权力集中在其总部和首席执行官手中。现在我们已经定义了分散化意味着什么，让我们开始深入研究如何在加密中使用它。

# 秘密分散化

如上所述的分散银行业务的例子，在加密中分散背后的思想是分配权限，使加密工作被分配，但也在权限上分散。Crypto 使用区块链来分散权限，这可以通过使用比特币作为例子来概述。

区块链只不过是一个记录硬币交易的账本。您可以将每个块视为分类账中的一页，在开始记录下一页(即块)的交易之前，需要对每页进行验证。正如我们将看到的，使区块链工作的系统的每个部分都是分散的。首先，区块链信息的存储是分散的。任何拥有计算机和互联网连接的人都可以运行一个接收交易并将其存储在区块链上的节点，只要他们的计算机足够强大，能够运行代码，存储所有交易历史，并且互联网连接足够快，能够处理每秒发生的交易数量。节点还跟踪钱包余额，并确保同一比特币不能被花费多次。您可以在 [bitnodes.io](https://bitnodes.io/) 上查看当前为 Bitcon 运行的节点数量。

作为验证事务的一部分，挖掘器解决一个数学问题，使节点能够在更新区块链之前验证每个块。对于他们的工作，矿工会得到一些硬币作为奖励。就像节点一样，只要有合适的计算机高效解决数学问题，任何人都可以开始挖掘。你可以在 chainbulletin.com 查看[比特币矿工地图。](https://chainbulletin.com/bitcoin-mining-map/)

由于区块链及其操作的分散性，其用户可以获得多种好处。首先，你不能指责任何人。即任何人都可以发送/接收 Bitcon，参与比特币的运营(节点和矿工)。创建虚假事务或阻止事务的唯一方法是拥有 50%以上的挖掘能力或运行 50%以上的节点。第二，任何人都可以查阅区块链所有交易的全部历史。你可以运行一个节点，比特币软件就会下载整个区块链。这意味着很容易验证哪个钱包从哪个钱包收到了多少钱，并检查任何钱包的余额。由于这两个性质，系统不需要信任任何一个权威，即无信任系统。

操作区块链的分布式授权是“非集中化”一词在密码学中的主要用法。可以想象，去中心化已经扩展到让其他‘事情’去中心化了。让我们进一步探索这个术语是如何演变的。

# 权力下放的形式

可以想象，分权的概念已经应用到更多的领域。实际上，更准确地描述去中心化的方式是区块链技术已经被应用到这些领域。首先，随着以太坊中智能合约的引入，应用程序(DApp)被去中心化。创新之处在于，您现在可以在区块链中运行代码，因此智能合约的存储和执行是分散的。现在，应用程序的哪一部分是分散的取决于应用程序开发人员。您可以存储帐户信息、数字版权或数字商品的唯一所有权(NFT)。所有这些都可以与交易联系起来，即兑换硬币以获得访问权限。这样，所有的信息都可以分散。这样做的好处是没人能拿走你买的东西。然而，如果 DApp 开发商停止经营，那么你买的东西将一文不值。

第二，crypto 是去中心化金融，去中心化金融(DeFi)。您的基本服务，如贷款、保证金、期权和其他金融产品，可以通过区块链完成，而不必通过中央机构。使用智能合约时，条款和金额会存储在使用合约的区块链中。任何与金融产品有关的操作都要经过合同。这意味着所提供的服务是分散的，也就是说，任何拥有可以与智能合约交互的钱包的人都可以使用这些服务。

第三，分散交换(DEX)是 crypto 分散交换的地方，是 DeFi 的一部分。在比特币基地这样的传统交易所，中央机构为你保管你的硬币。当您买卖硬币时，比特币基地匹配买卖订单，并将适当的硬币/法定货币存入您的账户。相比之下，用户可以用一个特定的密码交换另一个没有中央授权的密码，如 Uniswap。用户可以使用智能合约直接与另一个用户交换硬币。

最后，分散自治组织(DAO)正在分散组织。他们没有任何领导或组织。任何人都可以加入，通常是通过购买治理令牌。这些令牌使所有者能够对 DAO 感兴趣的主题进行投票。有时候一个道会凑在一起办一个达克斯，一个区块链或者买一些艺术品

# 结论

作为一个新的密码爱好者，试图理解行话在开始时可能会令人生畏。然而，crypto 只是一种看待现存社会契约的新方式。如果你能把密码术语和现有系统(如央行和政治系统)联系起来，所有这些概念都会变得更容易理解。例如，权力的分散类似于从一方面的独裁(中央集权制)到一方面的民主(分权制)。在加密中，区块链的操作是分散的，使得任何人都可以参与保持区块链的运行。然而，一个分散的系统可以成为一个集中的，如果一个当局获得了一个集中系统的大部分。如果一个权力机构持有一个 DAO 的多数股权或拥有超过 50%的采矿权力，则权力集中到该权力机构。一个著名的例子是大型互联网公司。总的来说，互联网是一个去中心化的系统，在这个系统中，没有一个权威机构来维持互联网的运行。对于谷歌、脸书和亚马逊等热门应用，我们与互联网的互动方式已经集中化，因为它们往往会获得最大的流量，因此会影响我们在互联网上的搜索内容、连接方式和购物方式。因此，我们生活在一个有趣的时代，有些力量试图将互联网集中化，而另一些力量则试图将其分散化。所以，我们在另一边见，来谈谈这件事的结果吧！
