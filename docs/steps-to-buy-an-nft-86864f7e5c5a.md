## 设置钱包的初学者指南

# 介绍

NFT 的，或不可替代的令牌，是区块链上托管的所有权凭证。这些可以是图像、音频或视频的形式。

在 2021 年的过程中，NFT 已经爆炸，著名的品牌和名人创造和购买 NFT 的，过去一年的数量在 NFT 土地上呈指数上升。我预计 2022 年将推动 NFT 走得更远。

在购买 NFT 的指南中，我们将回顾最关键的步骤:

-做你的研究

-购买密码

-获取加密钱包，如元掩码

-将加密发送到钱包

-将钱包同步到市场，如 Opensea

-买一辆 NFT

# 做你的调查

这是购买 NFT 的过程中重要的一步，但也是经常被遗忘的一步。

做你的研究。

知道你买的是什么。“你是在考虑购买具有艺术价值的东西，还是想买一幅 NFT 作品成为社区的一部分，并获得与持有相关的效用,”NFT 说。

无论你在寻找什么，确保你了解购买 NFT 的风险。你要知道谁在创造 NFT，他们的目标是什么。

# 2:购买密码

为了这个指南，让我们现在坚持以太坊。目前，大多数 NFT 人生活在以太坊区块链上，这是 NFT 最大的社区，也是最希望创建他们的项目并在上面销售 NFT 的社区。

要获得以太坊，你必须从加密货币交易所购买，如比特币基地或币安。比特币基地可能会更容易，如果这是你第一次基于他们的用户友好的设计。

并非所有交易所都出售每一种加密货币，但作为市值第二大的交易所，以太坊将出现在大多数(如果不是所有)交易所。

有以太坊的替代品，如 Solana，Tezos 和基于 NFT 的 Cardano，但目前，我们将坚持以太坊。

确保你有足够的以太币来支付油费，我会在第 6 步解释。

# 3:买一个加密钱包

你需要一个加密钱包来存放你所有的 NFT 和加密货币，对于以太坊，我推荐 Metamask。它对用户非常友好，即使不是全部，大多数网站都有元掩码，可以很容易地连接。Metamask 是一个浏览器插件和扩展。谷歌 Chrome 和 Brave 浏览器与它配合得最好。

通过他们的网站 [Metamask.io](http://Metamask.io) 下载安装 Metamask。它需要一秒钟来加载，然后要求您创建一个密码。这样做，你会收到你的秘密恢复短语，或你的种子短语。这是一个 12 个单词的短语，可以用来从任何地方访问你的帐户，所以把它写在一张纸上，并把它放在一个安全的地方。不要把这个给任何人，因为这将允许他们完全访问你的 Metamask 钱包。很多骗子试图从你这里得到这个，但是只要你不把你的种子短语给他们，你应该是安全的。

如果你想在其他区块链上买 NFT 的，我推荐索拉纳的幻影钱包和特佐斯的 Kukai 钱包。

# 4:向您的钱包发送资金

这一部分很简单，但是很多人在第一次做的时候会被吓到。

去你的交易所，找到“发送”按钮。弄清楚你想往你的钱包里存多少钱。在它询问你把它送到哪里的地方，那是你放置你的公共地址的地方。

打开你的 Metamask 钱包，找到你的以太坊公共地址。在元掩码中，它应该以 0x 开头，并显示在您的帐户名下。点击它复制地址，并粘贴到交换。该地址是您的钱包所独有的，类似于电子邮件地址或银行账号。

其他钱包也是如此。

# 5:将钱包同步到市场

找到你想购买 NFT 的市场，我们将使用 [Opensea.io](http://Opensea.io) 因为它是最大的以太坊 NFT 市场。在右上角，你会看到一个钱包图标，点击它，并连接到 Metamask。

这应该会将你的钱包链接到该网站，你可以设置自己的收藏，并查看你的个人资料，显示连接时你所有的 NFT。

还有其他市场，比如 Rarible.com 的，这是我在 NFT 风味标志项目中使用的市场。

# 6:买一辆 NFT

环顾网站，你可以探索许多收藏，并找到你想买的 NFT。点击一个 NFT 后，你可以找到它的收藏，通常有创作者的 twitter，网站和 discord 的按钮。把这些用于你自己的研究。

一旦你决定要买什么，你可以点击它，然后点击“立即购买”以标价购买或“出价”，选择为 NFT 提供什么，这就需要所有者在它通过之前接受它。无论你选择什么，要知道除了你买的东西之外，还会有一笔交易费，即汽油费。天然气的价格波动很大，可能很高，但是你可以在 etherscan.io/gastracker 的[等网站上查看当前的天然气价格。尝试在汽油价格较低时购买，以节省一些交易费用。对于新用户来说，这一部分可能会让人不知所措。](http://etherscan.io/gastracker)

也有 NFT 的作品被拍卖，你可以对这些作品“出价”，如果拍卖结束时你是出价最高的人，NFT 就是你的了！

值得注意的是，区块链如 Solana 和 Tezos 的燃气费明显低于以太坊，但没有以太坊区块链那么多用户。

# 结论

购买你的第一个 NFT 起初看起来势不可挡，但我希望这个指南能减轻你的一些担忧。一旦你买了一个或两个，你一定会很快掌握它的窍门，你可以真正开始在这个空间中享受乐趣。

值得注意的是，Solana 和 Tezos 等区块链人也患有 NFT 氏症，虽然这仍处于早期阶段，但交易期间需要支付的天然气费用少得令人难以置信，目前约为 1/100 美分，这是值得注意的。你需要一个幻影钱包在 Solana 网络上购买，一个 Kukai 钱包在 Tezos 网络上购买。

你可以使用这个指南，用 Phantom 或 Kukai 替换 Metamask，用 Solana 或 Tezos 替换 Ethereum，如果你愿意的话。

我希望你能在读完这本指南后轻松买到你的第一部 NFT。
