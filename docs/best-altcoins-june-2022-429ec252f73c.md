# 克罗诺斯(CRO)

对于 Crypto.com 平台的公用令牌 Cronos 去年实现了大幅增长，这主要归功于该平台积极的广告运营。CRO 代币可用于节省费用，以及为了有资格获得进一步的额外津贴而下注代币。Crypto.com 预付借记卡是另一个很好地利用代币的例子。
